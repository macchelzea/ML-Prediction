# ML-Prediction
Running some predictive analysis.

Ensure that all necessary packages are installed. 

## To run the code:
```bash
conda create -n venvMLOps python=3.12 -y

conda activate venvMLOps

pip install -r requirements.txt
```


## Workflow:

1. constants
2. entity
3. components
4. pipeline
5. Main file
